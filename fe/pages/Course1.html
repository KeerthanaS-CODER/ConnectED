Overview:

This 3-month mentor-led program is designed to help beginners master the fundamentals of web development using HTML, CSS, and JavaScript. Guided by industry expert Jody Homes, you’ll learn how to build dynamic, responsive websites from scratch, equipping you with the essential skills for a career in web development.

What You'll Learn:

  • HTML: Structuring web content with proper tags and semantic elements.
  • CSS: Designing beautiful, responsive web pages with advanced styling techniques.
  • JavaScript: Adding interactivity to your web pages, including event handling and DOM manipulation.
  • Project Work: Build your own portfolio-ready website by the end of the program.

Program Highlights:

  • Hands-On Learning: Work on practical projects and assignments designed to strengthen your coding skills.
  • One-on-One Mentorship: Direct guidance from Jody Homes, who will provide personalized feedback and career advice.
  • Interactive Sessions: Weekly live classes and Q&A sessions to discuss key concepts and clear any doubts.
  • Networking: Connect with like-minded peers and expand your professional network.

Preliminary and Supplementary Domains:

  • Preliminary Domains: HTML, CSS, JavaScript
  • Supplementary Domains: API integration, File handling, Networks, UI & UX design
